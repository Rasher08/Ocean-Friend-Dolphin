## 🚀 Как запустить игру (Инструкция)

Чтобы поиграть в "Ocean Friend" на своем компьютере, выполните следующие шаги:

### 1. Установка Python
Убедитесь, что у вас установлен **Python 3.10** или выше. 
* Проверить версию в терминале: `python --version`
* Если Python нет, скачайте его с [официального сайта](https://www.python.org/).

### 2. Подготовка файлов
1. Скачайте этот репозиторий кнопкой **Code -> Download ZIP**.
2. **Обязательно** распакуйте скачанный архив в обычную папку (не запускайте файл прямо из архива, иначе картинки не подгрузятся!).

### 3. Установка зависимостей
Игра написана на библиотеке **Pygame**. Если она не установлена, откройте терминал (командную строку) в папке с игрой и введите:
```bash
pip install pygame
```
### 4. Запуск
Просто запустите главный файл игры:

```bash
python main.py
```
### 🛠 Технические требования
ОС: Windows / macOS / Linux

Разрешение экрана: минимум 900x600

Все ассеты (папки 1-11) должны находиться в одной директории с файлом main.py.
